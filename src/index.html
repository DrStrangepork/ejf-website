About this website

First, I did not write the code that drives what you see on this website. I am not a website developer, so I used a template from HTML5 UP and filled in the necessary details. I did, however, build the infrastructure using Amazon Web Services. Cullancarey.com is a static AWS S3 website fronted by Cloudfront. Please see the architecture diagram below.

When I push changes to the Github repository, an AWS CodePipeline is kicked off. This pipeline extracts the necessary files for the website and pushes them to the corresponding S3 bucket. It also invokes a Lambda function to invalidate the Cloudfront cache, so the updates are available immediately. This website is highly available. The main website bucket replicates to the failover bucket in a different region. Access to the S3 bucket is restricted to the Cloudfront OAI. Terraform fully manages the website infrastructure. I used a Terraform module I built and published to the Terraform registry. It is available for use here. Please see this link for the Github repository. Building cullancarey.com was a fun project and provided a very valuable experience.
